# Heart-Disease-Prediction-
# Heart Disease Prediction using Logistic Regression

## Project Overview

This project focuses on predicting the presence of heart disease using machine learning techniques. The dataset contains patient health information such as age, cholesterol level, blood pressure, chest pain type, and other medical attributes. A Logistic Regression model was implemented to classify whether a patient is at risk of heart disease.

## Objectives

- Analyze patient health data
- Perform data cleaning and preprocessing
- Train a classification model for heart disease prediction
- Evaluate model performance using multiple metrics
- Visualize insights and model results

## Dataset

The dataset includes medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol Level
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Other cardiovascular indicators

Target Variable:
- `num`
  - 0 = No Heart Disease
  - 1 = Heart Disease Present

## Project Workflow

### 1. Data Exploration
- Loaded and inspected the dataset
- Checked data types and summary statistics
- Identified missing values
- Visualized key health indicators

### 2. Data Preprocessing
- Removed missing values
- Applied feature encoding where necessary
- Separated features and target variable
- Standardized features using StandardScaler

### 3. Model Development
- Split data into training and testing sets (80:20)
- Implemented Logistic Regression for classification
- Trained the model on processed data

### 4. Model Evaluation
The model was evaluated using:

- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix

### 5. Visualization
- Age Distribution Analysis
- Blood Pressure Distribution
- Cholesterol Distribution
- Confusion Matrix Heatmap

## Results

The Logistic Regression model successfully classified patients based on their health records and demonstrated effective performance for heart disease prediction. Evaluation metrics and visualizations were used to assess the model's reliability and predictive capability.
